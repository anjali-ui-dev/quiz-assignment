Quiz Assignment Using ReactJS


Git clone: https://github.com/anjali-ui-dev/quiz-assignment.git
Run npm install in project folder
Run `./node_modules/.bin/webpack --progress ` for build project
Run `./node_modules/.bin.webpack-dev-server` from build project & run server
Hit `localhost://8080` from you url to run project on your browser



It's a quiz assignment which fetch questions from server and allow user to attempt questions one by one.
User has a facility to go on previous question if missed to attempt by mistake or not able to solve on partcular time.
After attempting 20 questions, user have to confirm the submission and at the end user will get total score plus list to correct answers.
